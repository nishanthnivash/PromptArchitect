# PromptArchitect

Turn raw thoughts into high-performance AI instructions.

## Repository
[https://github.com/nishanthnivash/PromptArchitect](https://github.com/nishanthnivash/PromptArchitect)

## Features
- **Raw Thought Input**: Natural language processing for messy ideas.
- **AI Intent Detection**: Identifies core objectives using Gemini.
- **4-Style Architecture**: Generates Persona, Chain-of-Thought, Technical, and Marketing prompts.
- **Score Reasoning**: Explains why prompts are rated for Clarity, Specificity, and Quality.
- **Library**: Save and star your favorite architectures.

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Set your environment variables (Firebase & Gemini API)
4. Run development server: `npm run dev`
