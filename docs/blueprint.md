# **App Name**: PromptArchitect

## Core Features:

- Raw Thought Input: A clean text input area for users to enter their initial idea or raw thought.
- AI-Powered Intent Analysis: Utilize an LLM (e.g., Gemini) as a tool to analyze the user's raw thought, categorizing its intent, identifying potential personas, and inferring the target audience.
- Four-Style Prompt Generation: Based on the AI analysis, automatically generate four distinct prompt variations: Persona, Chain-of-Thought, Minimalist/Direct, and Creative/Brainstorming.
- Prompt Display and Review: Present the four generated prompts in an organized and easy-to-read format, allowing users to review each one.
- Prompt Editing: Enable users to directly edit or tweak the generated prompts to better suit their specific needs.
- Copy to Clipboard: Provide a simple one-click function to copy any of the generated or edited prompts to the clipboard for immediate use.

## Style Guidelines:

- Color Anchor: 'Structure', 'Innovation', 'Clarity'. A light color scheme for readability and focus. Primary color (violet-indigo): #553399. A deep, intellectual hue that conveys thoughtfulness and insight, providing good contrast against a light background. Background color (light violet): #F7F5F9. A very light, desaturated variant of the primary color to maintain visual coherence and provide a subtle warmth. Accent color (bright blue): #3E8BE6. A vibrant and clear blue, analogous to the primary, for calls-to-action and key highlights, promoting engagement.
- Body and headline font: 'Inter' (sans-serif). Chosen for its modern, neutral, and highly readable qualities, making it ideal for displaying structured textual content and prompts clearly.
- Utilize clean, minimal line-art icons for actions such as 'copy', 'edit', and 'generate', aligning with a modern and efficient digital tool aesthetic.
- Employ a single-page, responsive layout. The user input field should be prominently featured at the top, with generated prompts appearing clearly beneath it in distinct, well-organized sections to facilitate comparison and selection.
- Implement subtle animations for user feedback, such as a gentle loading state during AI generation and a quick visual confirmation upon copying a prompt, to enhance perceived responsiveness without distraction.
